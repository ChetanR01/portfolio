Blog Website

  • Tech Stack: Python, Django, HTML, CSS, Bootstrap, JavaScript, GCP, Postgresql, SMTP, Heroku
  • Github URL: Project Link

This is a Blog Website containing blogs of different categories and domains. I have developed this website to practice my Django skills. The website front-end template is downloaded from TemplateMo. I did some changes in the template required for this website. And added all Backend related stuff using Python-Django.

Featues:

  • Category Wise Blogs
  • Tags Wise Blogs
  • Easy Navigation.
  • Email Subscription.
  • Easy to Share on Social Media
  • Commenting Option
  • Dynamic Presentation
  • Trending Blogs on Home Page
  • Search For Blog Post
  • Proper Pagination
  • User Login/SignUp
  • Logged-in User Profile view and update
  • Contact Form
  • Subscription By Categories and Tags
