I have developed software to do 12 number conversions, 8 ASCII conversions, Binary Arithmetics, and 1's and 2's complements. Which shows all steps to calculate the respective operation. I used Tkinter Library for the GUI of the software

Features:

  • Do all 12 types of Number System Conversion.
  • Finds the ASCII value of all four Number Systems.
  • Convert ASCII Value to all four Number Systems.
  • Do all four basic Arithmetic Operations on Binary Numbers.
  • Find One's and Two's Complements.
  • Do all the above-listed operations with explanations and steps.
