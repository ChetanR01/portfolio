Wallet System API

  • Tech Stack: JavaScript, Nodejs, Sequelize, ORM, JWT, JSON, Postgresql, ExpressJs, Git, GitHub
  • Github URL: Project Link

A wallet system is a secure and versatile wallet system API using Node.js. This API allows users to manage their digital funds, conduct various transactions (deposits, withdrawals, transfers), and view transaction histories. The API also offers user-friendly endpoints, documentation, and a responsive interface for seamless interaction.

Features:

  1. User Registration and Authentication: Allow users to create accounts with email and password. Implement secure authentication using JWT or OAuth.
  2. Wallet Creation and Management: Display wallet balances and transaction history.
  3. Transaction Types: Support various types of transactions: deposits, transfers, and balance checks. Allow users to specify transaction details, such as recipient and amount.
  4. Transaction status and statement: Get transaction status and Fetch all transactions past transaction statements.
