SmartCart Grocery Store Web Application

  • Tech Stack: Python, Django, HTML, CSS, JavaScript, Bootstrap, Postgresql, AWS
  • Github URL: Project Link

Developed a comprehensive grocery store website featuring three distinct portals: Main Customer Website, Store Admin, and Delivery Portal.

✔ Main Customer Website: Designed an intuitive and user-friendly interface for customers to conveniently browse, search, and purchase grocery items online. Implemented features such as personalized recommendations, user account management, and seamless checkout process.

✔ Store Admin Portal: Built a robust admin portal for store administrators to efficiently manage operations. Enabled them to update product information, handle inventory management, track orders, and respond to customer inquiries. Implemented sales reporting, promotions management, and customer feedback features.

✔ Delivery Portal: Developed a dedicated delivery portal for optimizing delivery logistics. Streamlined delivery scheduling, order tracking, and route optimization for delivery personnel. Enabled effective communication with customers regarding their deliveries.
